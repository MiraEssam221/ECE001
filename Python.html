3- Python

Python is a complex, object-oriented programming language based on versatile and powerful functionality.
It was built at CWI in the Netherlands in the 1980's and published for the first time in 1991.
Python can act as a coding or glue language swiftly for integrate systems.
This is also suitable for designing the Rapid Application Develop(RAD) software.
Python is a (bytecode-compiled) flexible, programming language. No variable, input, process or procedure types are declared in the source code.
This keeps the code simple and quick, and the testing of the source code compilation time is lost.
Python monitors all sorts of runtime variables and flags of code that cause error while running. It's simple for inputs and users to work on.
In addition, if the code is attempting to read a parameter which is not an identified value, the processor produces or creates a runtime error in Python.
  Python is case-sensitive so that "a" or "A" isn't similar. The ending of a line represents the completion of a phrase, at the end of a statement Python doesn't need a semicolon.
Statements start with "#" and continue to the end.
An unusual feature of Python is that a coding piece's whitespace influences its meaning.
The same coding should be set by the indentation of their relative component or if or anything else, as part of the reasonable chain of statements, such as those that forms the mechanism.
When one of the lines in a category has a distinct indentation, a syntax error is marked.
