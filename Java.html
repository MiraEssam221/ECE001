1- Java

Java is a high-level, object-oriented programming language with many capabilities, making it perfect for web-based development.
In Sun Microsystems (American company) Java was initially called Oak and built up in 1990 to add C++ language skills.
The WORA concept (Write Once Run Anywhere) was used in the development of Java.
In 1995, the language became global and is now owned by Oracle.
It is specified by the programming language, a compilation processor, a core library and a runtime (virtual Java machine).
Java runtime enables users to create code of programming in languages besides Java, if it is still running on the virtual machine.
The java platform has the following features: 1- Unique framework: java applications are relative to Java's virtual machine and do not have clear access to the operating system.
This renders extremely flexible java applications.
2- Object-oriented programming language: All the components in Java are structures, with the exception of basic data types.
3- Language that is  interpreted and assembled:Java source code is translated to a non-target application bytecode type.
The Java Virtual Machine (JVM) interprets these Bytecode orders. The JVM includes a Hotspot-Compiler which converts essential bytecode output orders for the native code.
4- Instantaneous storage devices: Java handles the assignment and decomposition of memory in order to create new structures.
The software has no immediate memory control. A so-called trash collector immediately erases components at which no useful link takes place.
